# Downloader-98

Downloader-98 is a retro-inspired web app built with React, TypeScript, and Vite. It emulates a Windows 98 desktop experience for uploading, downloading, and managing files using a PIN-based system.

## Getting Started

### Prerequisites

- Node.js (v18 or newer recommended)
- npm

### Installation

```bash
git clone https://github.com/apaluchdev/downloader-98.git
cd downloader-98
npm install
```

### Running the App

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Building for Production

```bash
npm run build
```

## Project Structure

- `src/` - Main source code
  - `components/` - UI components (windows, icons, player)
  - `hooks/` - Custom React hooks
  - `types/` - TypeScript type definitions
- `public/` - Static assets
